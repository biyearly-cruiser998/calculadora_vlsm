## 🖥️ What this app does

**calculadora_vlsm** is a web app that helps you work with IP networks in a simple way.

You can use it to:

- split networks into subnets
- calculate host counts
- view IP addresses in binary
- check IP class
- see if a network is private or public
- study subnetting and VLSM with clear visuals

It is built for people who want a fast way to check network data without doing each step by hand.

## 🧠 Help for first-time users

If you are new to subnetting, start with one simple IP address and one network size.

A good order is:

1. enter the IP
2. check the class
3. review the host count
4. inspect the binary view
5. compare the subnet result
6. move on to VLSM

This makes it easier to understand how the values change.

## 🔐 Private and public IP ranges

The app helps you spot common private ranges such as:

- `10.0.0.0` to `10.255.255.255`
- `172.16.0.0` to `172.31.255.255`
- `192.168.0.0` to `192.168.255.255`

It also helps you see when an address is outside these ranges and is more likely to be public
