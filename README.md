# Calculadora VLSM Profesional

Esta página web es una **calculadora de VLSM (Variable Length Subnet Masking)** interactiva, diseñada para ingenieros de redes y estudiantes que necesitan segmentar redes IP de forma rápida y visual. 

La calculadora permite:

- Introducir una IP base y definir múltiples subredes con número de hosts deseados.  
- Calcular automáticamente: dirección de red, broadcast, primer y último host, hosts utilizables y sobrantes.  
- Visualización de la **representación binaria** de IP, máscara y broadcast, con los primeros tres octetos en azul apagado y el último en blanco, tabulada para mejor legibilidad.  
- Identificación de **tipo de red** (pública/privada) y **clase de IP** (A, B, C).  
- Mostrar todas las subredes calculadas en una **tabla de referencia CIDR**, y explorar los hosts individuales de cada subred mediante un selector.  
- Funcionalidad de desplegar/colapsar secciones con animaciones suaves.  
- Scroll automático en la tabla de hosts si supera 20 filas para facilitar la visualización.  
- Estilo moderno dark mode con badges, blur y diseño limpio.  
- Uso de **Hack Nerd Font** para todos los textos y compatibilidad con iconos de NerdFonts.  

La página está pensada para ser **responsive y profesional**, ideal para demostraciones o uso educativo, alojada fácilmente en **GitHub Pages**.
